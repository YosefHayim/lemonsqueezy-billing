# Event Types

**Source**: https://docs.lemonsqueezy.com/help/webhooks/event-types

## Attributes

| Property | Description |
|----------|-------------|
| `single-payment` | order_created - This event lets you know when new orders are placed. |
| `subscriptions` | subscription_created - This event lets you know when new subscriptions are created. |
| `events-sent-during-a-subscriptions-lifecycle` | Different events will be triggered at various events during a subscription’s life. |
